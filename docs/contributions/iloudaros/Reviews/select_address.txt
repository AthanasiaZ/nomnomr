Διορθώσεις που προτείνω για το Robustness Diagram "customer-select-address"
Οι Διορθώσεις που σημειώνονται μέσα σε παρενθέσεις είναι προαιρετικές.

- Αντικατάσταση των βελών που βγαίνουν από τον Πελάτη με επεξήγηση των εξερχόμενων βελών στο εκάστοτε boundary object.
- (Τα control objects που κάνουν fetch πληροφορίες που χρειάζεται ένα boundary object, να γενιούνται από αυτό το boundary object, ώστε να φαίνεται ξεκάθαρα ότι αυτό κάνει την κλήση των πληροφοριών.)
- Το επιστροφή στην αρχική οθόνη, αντί για control object, θα μπορούσε απλά να είναι ένα βέλος προς την Αρχική Οθόνη
- Το Boundary Object "Εισαχθείσες Διευθύνσεις" δεν κάνει fetch από πουθενά τις ήδη υπάρχουσες διευθύνσεις. Ίσως χρειάζεται να βάλουμε ένα Entity Object "Customer"
- Το βέλος Εύρεση - Σύστημα Χαρτών είναι ανάποδα
- (Να αντικατασταθούν οι βούλες με τα emoji φαγητού που υπάρχουν ήδη στις ροές στα Use Cases)
